# React + Vite

Bu projede, axios ile JSON-server kütüphanelerini kullandım. Projenin arayüzü için çok uğraşmadım, zaten amacım bu yeni öğrendiğim kütüphanelerde kendimi denemekti.


FAKE API'ı ayağa kaldırmak için kodların bulunduğu dizine terminalle girip "npx json-server --watch src/Quotes/Quotes.json" yazmanız yeterli.
